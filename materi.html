Jaringan Komputer

Perangkat jaringan komputer adalah alat yang digunakan untuk menghubungkan beberapa komputer ke dalam sebuah jaringan, sehingga memungkinkan komputer-komputer tersebut untuk saling berkomunikasi dan berbagi data. Beberapa contoh perangkat jaringan komputer meliputi router, switch, hub, repeater, bridge, gateway, dan firewall.

Router

Router adalah perangkat jaringan yang menghubungkan berbagai perangkat dengan internet dan mengatur paket data yang dikirim dan diterima. Dengan router, kita dapat menghubungkan perangkat seperti smartphone, laptop, tablet, dan lainnya ke layanan internet. Berikut beberapa hal yang perlu diketahui tentang router.

fungsi utama Router yaitu mengelola lalu lintas antar jaringan dan membagikan koneksi internet ke beberapa perangkat lain. Selain dua fungsi tersebut, ada beberapa fungsi lain yang dimilikinya, antara lain Menghubungkan jaringan ke DSL, Mentransmisikan informasi,Membaca alamat IP,Menyaring paket data,Menghubungkan jaringan.

Cara kerja router

router menghubungkan perangkat dalam jaringan dengan meneruskan paket data di antara mereka. Data ini dapat dikirim antar perangkat, atau dari perangkat ke internet. Router melakukan ini dengan menetapkan alamat IP lokal untuk masing-masing perangkat di jaringan. Hal ini untuk memastikan bahwa paket data berakhir di tempat yang tepat.

Router wired adalah perangkat yang biasa digunakan oleh pengguna internet rumahan untuk menghubungkan perangkat-perangkat mereka dengan internet menggunakan kabel LAN/Ethernet.

Router wireles

Wireless router adalah perangkat nirkabel yang digunakan untuk menghubungkan beberapa komputer agar terkoneksi dengan internet. Jika Anda ingin memelihara jaringan Wi-Fi untuk browsing dan melakukan remote working dari rumah, gunakanlah wireless router.

Core Router

Core Router ini biasanya digunakan oleh perusahaan dengan volume pengiriman paket data yang tinggi. Core router beroperasi pada inti jaringan dan tidak berkomunikasi dengan jaringan eksternal.

Edge Router

Edge router adalah perangkat berkapasitas lebih rendah yang ditempatkan pada batas jaringan. Hal ini memungkinkan jaringan internal untuk terhubung dengan jaringan eksternal.Router ini juga disebut sebagai access router karena menggunakan BGP eksternal (Border Gateway Protocol) untuk menyediakan konektivitas dengan jaringan jarak jauh melalui internet.
